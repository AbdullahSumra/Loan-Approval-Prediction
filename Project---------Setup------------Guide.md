\------------------------Project Setup Guide------------------------



1\. Project Folder Open Karna



PowerShell ya Terminal open karo.



Command likho:



cd "D:\\Student Performance Prediction"





2\. Requirements Install Karna



Project folder ke andar yeh command run karo:



pip install -r requirements.txt           ***----------->  guide for new system***





Is se:

\- Flask install ho ga

\- Pandas install ho ga

\- Scikit-learn install ho ga

\- Matplotlib install ho ga

\- Aur baqi required libraries install ho jayengi





3\. Agar Libraries Pehle Se Installed Hon



Agar koi library pehle se installed ho to pip dobara install nahi kare ga.



Terminal mein message aaye ga:



Requirement already satisfied



Yani:

\- Error nahi aaye ga

\- Dobara unnecessary install nahi ho ga





4\. Virtual Environment (Recommended)



Agar project professional tareeqe se chalana ho to virtual environment use karo.



Virtual environment create karne ke liye:



python -m venv venv





5\. Virtual Environment Activate Karna



Windows mein:



venv\\Scripts\\activate



Activate hone ke baad terminal ke start mein (venv) likha aaye ga.





6\. Flask App Run Karna



Project run karne ke liye command:



python app.py





7\. Localhost Open Karna



Browser mein yeh link open karo:
